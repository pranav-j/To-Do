
# ToDo

This project is a basic ToDo app built in React.js


## Deployment

To deploy this project, make sure toinstall Node.js, then clone the repo

```bash
  git clone https://github.com/pranav-j/To-Do.git
```

```bash
  npm install
```
```bash
  npm run dev
```
Now the app must be running on

http://localhost:5173/
